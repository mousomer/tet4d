# Plans Directory Guide

`docs/plans/` contains two kinds of documents:

## Active plans

- `topology_playground_current_authority.md`:
  current architecture authority for topology-playground behavior and migration
  precedence.
- `topology_playground_shell_redesign_spec.md`:
  active visible-shell redesign spec for the current topology-playground phase.

## Living reference docs

- `cleanup_master_plan.md`
- `piece_transform_extraction.md`
- `piece_transform_inventory.md`
- `topology_explorer_latency_pass1.md`

Historical execution reports and retired one-off plans live under:

- `docs/history/plans/`
- `docs/history/topology_playground/`
