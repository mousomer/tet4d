# tet4d Migration Bundle

This directory is generated by `tools/migration/export_config_bundle.py`.
It is a disposable migration artifact, not a source of truth.

Python remains the semantic oracle. Authoritative config stays under
`config/`, authoritative traces stay under `migration/golden_traces/`, and
authority docs stay in `docs/` plus `CURRENT_STATE.md`.

Unity/Godot replay spikes should consume this bundle as input data for trace
replay before implementing independent topology, gameplay, or endgame logic.
Engine scene, inspector, or project defaults must not replace the Python/config
authorities.

Regenerate and check with:

```bash
PYTHONPATH=src .venv/bin/python tools/migration/export_config_bundle.py --out migration/exported_bundle
PYTHONPATH=src .venv/bin/python tools/migration/export_config_bundle.py --check
PYTHONPATH=src .venv/bin/python tools/migration/compare_config_bundle.py migration/exported_bundle
```
